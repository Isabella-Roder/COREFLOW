# Roadmap - CoreFlow

## Fase 1 - Fundacao

- Criar projeto Spring Boot.
- Criar projeto Flask.
- Configurar banco de dados.
- Definir entidades iniciais.
- Criar README e requisitos.

## Fase 2 - Empresas e Usuarios

- Cadastro de empresas.
- Cadastro de usuarios.
- Login.
- Perfis e permissoes basicas.

## Fase 3 - Cadastros Operacionais

- Clientes.
- Fornecedores.
- Produtos.
- Servicos.

## Fase 4 - Estoque, Vendas e Compras

- Entrada e saida de estoque.
- Registro de vendas.
- Registro de compras.
- Atualizacao automatica do estoque.

## Fase 5 - Financeiro

- Contas a pagar.
- Contas a receber.
- Fluxo de caixa.
- Baixa de pagamentos e recebimentos.

## Fase 6 - Flask e Dashboards

- Telas administrativas com Flask.
- Dashboard gerencial.
- Consumo da API Spring Boot.

## Fase 7 - Analytics

- Relatorios com Python.
- Graficos.
- Exportacao CSV/PDF.
- Indicadores financeiros.
